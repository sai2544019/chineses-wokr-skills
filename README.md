# 中文网文写作 Skill 与《北风裂鼎》

这个仓库包含两部分：

- `chinese-webnovel-writing/`：中文网文写作 Codex skill，面向 300-500 章长篇连载、分卷、伏笔、设定圣经、去 AI 味和改稿。
- `six-towns-to-sui/`：使用该 skill 生成的长篇历史权谋小说工程，暂定名《北风裂鼎》，从北魏六镇之乱写到杨坚代周建隋。

## 目录

- `chinese-webnovel-writing/SKILL.md`：skill 主入口
- `chinese-webnovel-writing/references/`：长篇框架、设定圣经、流程、压力测试等参考文件
- `six-towns-to-sui/00-总纲.md`：500 章总纲
- `six-towns-to-sui/01-设定圣经.md`：小说设定圣经
- `six-towns-to-sui/02-第一卷细纲.md`：第一卷 50 章细纲
- `six-towns-to-sui/03-第一章-雪夜粮仓.md` 到 `13-第十章-城门开.md`：已生成正文
- `six-towns-to-sui/04-章节追踪表.md`：章节追踪表
- `six-towns-to-sui/14-前十章小结.md`：前十章阶段小结
