# 设定圣经

## 1. 不可改项

- 起点：北魏六镇之乱前后
- 终点：581年杨坚受禅，建立隋朝
- 核心主题：乱世不是英雄独造，而是无数人的求活和妥协推成的结果
- 叙事骨架：六镇边军崩坏 -> 北魏解体 -> 尔朱入洛 -> 东西魏分裂 -> 关陇集团成形 -> 北周灭齐 -> 杨坚代周
- 史实原则：大事件尊重史实，小人物用于穿针引线

## 2. 主线虚构家族

### 贺兰砺

- 身份：怀朔镇低级军户，第一代主视角
- 表面目标：让一家人在饥荒和军法中活下去
- 深层欲望：证明边镇军户不是洛阳可以随意牺牲的牲口
- 伤口：父辈为魏守边，却没有换来尊严
- 底线：不杀妇孺，不卖同袍
- 破底线条件：家人被卷入军法和饥荒
- 结局方向：死于乱军或尔朱体系早期清洗，给第二代留下血债

### 贺兰烬

- 身份：贺兰砺之子，第二代主视角
- 表面目标：在军头之间活下去并替父报仇
- 深层欲望：摆脱流民身份，成为能决定别人命运的人
- 伤口：亲眼看见父辈的忠诚毫无价值
- 底线：不再相信空名分
- 风险：会逐渐变成自己恨过的军头
- 结局方向：入关中，成为西魏/北周军府家族的奠基者

### 贺兰昭

- 身份：贺兰烬后人，第三代主视角
- 表面目标：保住家族在北周权力重组中的位置
- 深层欲望：弄清楚祖辈用血换来的秩序到底值不值
- 伤口：生在新秩序中，却背负旧乱世的债
- 底线：不愿再让家族成为权力棋子
- 风险：最终发现自己也只能在棋局里选择更少的坏
- 结局方向：见证杨坚受禅，成为全书收束视角

## 3. 历史人物功能

- 破六韩拔陵：边镇怒火的点燃者
- 尔朱荣：用屠刀重建秩序的人，也是旧秩序彻底死亡的标志
- 高欢：把六镇残部变成东魏/北齐军政机器的人
- 宇文泰：把关中变成制度实验场的人
- 独孤信：婚姻政治和关陇门阀网络的代表
- 宇文护：权臣政治的高压形态
- 周武帝宇文邕：北周集中权力并灭齐的关键君主
- 杨坚：乱世结构筛选出的收束者

## 4. 权力关键词

- 粮：六镇崩坏的第一把火
- 兵：乱世里最硬的通行证
- 名分：皇帝、宗室、权臣互相撕咬的外衣
- 婚姻：关陇集团内部的权力缝合
- 军府：从流民到国家机器的制度通道
- 血债：三代人的隐秘主线

## 5. 文风要求

- 不写教科书式历史介绍
- 不让人物替作者讲史
- 每个历史大事件都要落在具体人的命运变化上
- 权谋必须有物质基础：粮、兵、路、婚姻、官位、名分
- 群像人物说话必须有身份差异，边镇人、洛阳人、关中人不能一个腔调
